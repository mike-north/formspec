# @formspec/e2e

End-to-end and benchmark workspace coverage for FormSpec.

## Usage

Run the end-to-end suite from the monorepo root:

```bash
pnpm run test:e2e
```

Run the benchmark fixture directly:

```bash
pnpm --filter @formspec/e2e run benchmark:hybrid-tooling
```
